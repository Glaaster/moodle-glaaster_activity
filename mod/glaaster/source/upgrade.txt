This files describes API changes in /mod/glaaster/source/* - LTI source plugins,
information provided here is intended especially for developers.

=== 4.1 ===

* The callback get_shortcuts() is now removed. Please use get_course_content_items and get_all_content_items instead.

=== 3.9 ===

* The callback get_shortcuts() is now deprecated. Please use get_course_content_items and get_all_content_items instead.
  See source code examples in get_course_content_items() and get_all_content_items() in mod/glaaster/lib.php for details.

=== 3.1 ===

* Callback get_types() is deprecated, instead ltiglaastersource plugins can define callback get_shortcuts().
  See source code for lti_glaaster_get_shortcuts() and get_module_metadata().
